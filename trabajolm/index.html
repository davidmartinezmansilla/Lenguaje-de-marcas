Manual HTML Básico

Formato

A continuación vamos a ver de manera básica, lo que necesitamos para crear una pagina html.
Lo principal es darle un formato correcto para ello, lo primero que pondremos en nuestro
documento a guardar como .html, será <!doctype html>y además, le indicaremos
que tipo de idioma usará en caso de que se use un narrador para los invidentes
para ello justo debajo pondremos la etiqueta html con el atributo lang y especificaremos
el lenguaje que queremos, quedaria algo así <html lang="es">

El documento html, debe contener unas etiquetas obligatorias, entre ellas
podemos encontrar la html, como vimos anteriormente. Otra sería la etiqueta
<head> la cual sería la cabezera de nuestro documento, y dentro de esta,
podemos poner otras como la de <title>, justo después podemos poner
la de metadatos, la cual especificará que tipo de lenguaje de metadatos usará
en este caso <meta charset="utf-8">, las etiquetas de metadatos
no las veremos en la pagina web, por lo que en realidad, no importa donde las pongamos
Seguría la <body> el cuerpo del documento, donde irá el texto, imágenes,
listas, tablas... en fin, prácticamente todo lo que queramos añadir.
Esas serían las obligatorias a poner, las cuales, una vez pongamos, tendremos que cerrarlas
Para cerrarlas tendremos que volver a ponerlas, pero añadiendo un / antes del nombre de etiqueta
es decir, </head> </body> </html>

Dentro de algunas etiquetas, pueden ir por debajo otras, ya sea para añadir alguna función,
hacer que sea cursiva, etc... por ejemplo, dentro de <head> quizás nos interese poner
un título, para ello entre el rango de etiqueta head, usaremos la de title, quedando algo así.
<head>
<title>Título que quieras poner</title>
</head>

Como veís, todas las etiquetas que vamos abriendo, se van cerrando una vez acaban, para que
no haya problemas en el documento. Pero, no todas las etiquetas se cierran, algunas contienen
lo que se denomina como elemento vacío. Por ejemplo la etiqueta <br> es un salto de línea
y una vez se usa (abre) no hace falta cerrarla </br>, o <hr> para poner una línea vertical.

Títulos

Antes vimos como añadir un título en el <head>,pero ¿y si queremos añadir varios más?
Lo podemos hacer con las siguientes etiquetas por ejemplo, dentro de <body>:

<h1>Otro título</h1>

<h2>Otro título</h2>

<h3>Otro título</h3>

<h4>Otro título</h4>

<h5>Otro título</h5>
<h6>Otro título</h6>

Agrupación

Ya vimos como incluir párrafos, asíque iremos a otro tipo de agrupaciones, como por ejemplo, las listas.
A la hora de crear listas, tenemos dos opciones: una lista ordenada (por numeros) o bien sin ordenar.
Si queremos una ordenada, tendremos que usar la etiqueta <ol>(ol, de ordened list), si contrariamente
queremos usar una sin ordenar, usaremos esta otra etiqueta <ul> (unordered list).
Una vez creada, tocará poner los elementos de texto que precisemos en ella, para ello, hacemos uso de otra
etiqueta más llamada, <li> (list) e iremos cerrando todas ellas una vez terminemos, incluida la lista.
Debería quedarnos algo tal que así:

Ordenada
    <ol>
    <li>uno<li>
  1. uno
    2. <li>dos<li>
  2. dos
    3. <li>tres<li>
  3. tres
    4. </ol>
Sin ordenar
    <ul>
    <li>uno<li>
  • uno
    • <li>dos<li>
  • dos
    • <li>tres<li>
  • tres
    • </ul>

También tenemos el texto preformado. Esto lo podemos usar para cuando queramos escribir un texto en el que
nos interese que se muestre tal cual es, ya que se escribe monoespaciado. Tenemos que tener en cuenta
que si ponemos un salto de linea de manera normal como en un procesador de textos, o muchos espacios en blanco
la pagina web, lo tratará como si fuera un único espacio en blanco. El texto preformado se usa con <pre>
<pre>

Este texto se escribirá   	   			tal cual			
			gracias a esta
etiqueta que estamos usando	       	ahora mismo.
</pre>

Con <p> podemos crear párrafos, pero si añadimos saltos de linea de manera normal, no se verán reflejados

Elementos Semanticos

Para añadir resaltados, tendremos que usar las siguientes expresiones en el texto que queramos usarlas
Italic, o Bastardilla <i>, Bold, o Negrita <b> y Underlined, o Subrayada <u>
También podemos encontrar <em> para recalcar o dar énfasis, <strong> para una parte seria,
<small> letra pequeña, o <s> representando contenido irrelevante u obsoleto
Recordad, que estas se han de cerrar, sino, aplicarian sus efectos a toda la página.

A la hora de crear Hiperenlaces, para poder visitar otra pagina, se hará de la siguiente manera:

<a href="http://www.google.es">Google</a>
Google --> (así es como debería quedar)

Donde <a> sería anchor (ancla), href el atriburo para crear el enlace, y después ponemos
la dirección a donde queremos ir (ya sea local para ir a otro archivo, o una página web distinta.
Entre la etiqueta de apertura y la de cierre, podemos poner por ejemplo el nombre de a donde iremos
en caso de se use el Hiperenlace creado.

El salto de línea es para eso mismo, añadir un salto de línea, ya que como dijimos, anteriormente
la página web no lo mostrará como tal, ya que será un espacio en blanco, para ello utilizamos el <br>.

Tablas

Podemos añadir tablas a nuestra web, con la etiqueta <table>, después para añadir filas, <tr>
y para las columnas <td> (si queremos dentro de la etiqueta table, se puede poner el atributo border
para que le ponga un pequeño borde a nuestra tabla, de esta manera se visualiza mejor).

<table border>
	<tr>
		<td>esto<tr>
		<td>es<tr>
	</tr>
	<tr>
		<td>una<tr>
		<td>tabla<tr>
	</tr>
</table>
esto es
una tabla

Contenido Encajado: Imágenes

Para poder poner imágenes usamos la etiqueta <img> dentro de su etiqueta, podemos poner varios atributos,
el scr nos sirve para dar la ruta de la imágen que queremos colocar, le podemos poner un width, para cambiar su ancho
o un heigh, para la altura. No solo podemos poner imágenes del tipo .jpg, sino también .gif, .png, o otros.
A una imágen le podemos poner un texto alternativo para dar una descripción de la imágen con el atributo alt.

Los mapas de imágenes, se usan para añadir una zona en la imagen la cual sirve para convertirla en una imagen interactiva
es decir, dentro de una imagen, se pueden colocar varias zonas, las cuales al hacer click en estas, nos lleven a diferentes
enlaces, o bien zonas de la misma página o bien otras, incluso externas
Para esto tendremos que crear la etiqueta <map name="nombre del mapa"> y dentro de la etiqueta de <img> en la que
queramos añadir esto, pondremos el atributo usemap="#nombre del mapa" para que esten relacionados.

Entre la etiqueta map, y su cierre, podremos usar la etiqueta <area>, para poder indicar que tipo de área queremos usar
por ejemplo si usamos el atributo rect, (coordenada) esto formara una zona rectangular en la que podremos hacer click dentro de la imagen.
Al darle el atributo de coordenada, tendremos que especificar que zona de imagen es, y esto se vería así
rect: x-izq, y-supr, x-dcha, y-inf
circle: x-centro, y-centro, radio.
poly: x1,y1, x2,y2, x3,y3, etc...
